Данный скетч делает следующее: передатчик (TX) отправляет массив данных, который генерируется согласно показаниям с кнопки и с двух потенциомтеров. Приёмник (RX) получает массив, и записывает данные на реле, сервомашинку и генерирует ШИМ сигнал на транзистор.
